twainapp.win32.installer.msi
Use this MSI to install on 32-bit versions of Windows.  The default folder for the program
is "C:\Program Files\TWAIN Working Group\TWAIN 2.3 App Sample\"

twainapp.win64.installer.msi
Use this MSI to install on 64-bit versions of Windows.  The default folder for the program
is "C:\Program Files (86)\TWAIN Working Group\TWAIN 2.3 App Sample\" for the 32-bit version,
and "C:\Program Files\TWAIN Working Group\TWAIN 2.3 App Sample\" for the 64-bit version.
Note that not all scanner vendors offer 64-bit versions of their drivers.
